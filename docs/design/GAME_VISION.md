# Mežtirgus Game Vision

A realistic forestry company simulator.

The fantasy:
A player starts small and gradually builds a timber business.

The player manages:
- sourcing timber
- understanding quality
- choosing buyers
- managing transport
- controlling cash flow
- expanding assets

The game should feel like operating a real company, not clicking upgrades.

Important tensions:
- profitable deals can destroy cash flow
- cheap timber may have hidden problems
- distance changes economics
- information has value
- scale creates logistics problems
